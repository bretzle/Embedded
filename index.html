Embedded Manual

Introduction

This book can be read online at https://bretzle.github.io/Embedded/

This system was designed around the DE10-Lite FPGA, but it is probably possible to get it working on other boards. The system supports the controlling the following:

  • LEDs
  • Touch LCD display
  • Accelerometer
  • Pushbuttons
  • Switches
  • Servo motors
  • Analog joystick
  • Seven-segment displays

The system was designed using Intel-Altera's provided tools and IP. Primarily the NIOS II soft-core cpu.
